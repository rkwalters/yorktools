# YorkTools

R package for York regression with extended options (e.g. fixed intercept)

# Installation:

```
require(devtools)
devtools::install_github("rwalters/yorktools")
```

Currently requires a PAT (personal access token) to install from private repo.

# Usage

See `?york`
